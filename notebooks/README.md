# Notebooks

Two Jupyter notebooks are provided as tutorials in how to use _k223d_. Two tutorials are provided:

- **planar_eg** shows how to create a simple planar fault surface and put a stochastic slip distribution on it. 
- **EFSM_eg** is more comoplicated as it takes an initial mesh generated from the [European Databases of Seismogenic Faults](https://www.seismofaults.eu/), remeshes it an places a slip distribution on it. 

The folder  **local_py_scripts** contains python scripts that are used in the two tutorials. If running the notebooks keep this folder in the same relative place in relation to the notebooks in order that they will function correctly.
